FitVids.js example

FitVid maintains player embed size aspect ratio, using JS to apply RWD padding-top technique. More about FitVids.js
Also see kaltura's no-js RWD example

Note: Changes are needed for fitVid to work against kWidget.embed fork available here.

Sample FitVids.js with kWidget.embed Code: 
<script type="text/javascript" src="resources/FitVids.js"></script>
<div class="videoContainer" style="width:80%;border:solid thin black">
	<div class="fluid-width-video-wrapper">
		<div id="kaltura_player" style="width:640px;height:360px"></div>
	</div>
</div>
<script>
	kWidget.embed({
		'targetId': 'kaltura_player',
		'wid': '_243342',
		'uiconf_id' : '12868012',
		'entry_id' : '1_sf5ovm7u',
	});
</script>

Page content in dynamic width content column.
