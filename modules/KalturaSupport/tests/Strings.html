Player strings override

In this demo, player strings are being override by Flashvars using the "strings" plugin.
To override a string, define a new value for it's key.
For the player below, the following keys and values are specified:

  • "mwe-embedplayer-play_clip": "Click to play"
  • "mwe-embedplayer-pause_clip": "Click to pause"
  • "mwe-embedplayer-player_fullscreen": "Enter full screen"
  • "mwe-embedplayer-volume-mute": "Turn sound off"
  • "mwe-embedplayer-volume-unmute": "Turn sound on"
Roll-over the play, mute and full-screen buttons to see the new tooltip values.
