Sample AutoThemer "making player theming easy"

Works with existing html:
1) Design your player in dreamweaver / html5 editor of your choice around an html5 embed tag.
2) Add class names to your player elements ( ie centerPlayButton, playButton, pauseButton etc.
3) Call $j('#playerTarget').playerThemer( { ..custom options .. }). ( Custom options allow detailed customization of what is displayed when, more on that later ).

Once playerThemer is called your player is driven by kaltura html5 lib adding support for:
* player bindings, ( playhead updates, status updates, iPad overlays )
* player interface actions ( fullscreen, playhead scruber etc )
* analytics tools ( normalize html5 events across implementations for easy analytics events bindings )
* And lots of other kaltura and html5 related features / modules ( html5 subtitle timedText,
