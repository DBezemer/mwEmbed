Kaltura On-Page Chaptering Examples

This page provides samples for the chapters view plugin. You can view all the configuration options on the chapters integration page.

Full chaptering interface - Lean Horizontal Carrousel, custom render

Custom renders are easy with the chapterRenderer hook: 
<script>
		kWidget.embed({
			...
			flashvars: {
				"chaptersView": {
					...
					"chapterRenderer" : "onChapterRendererSample1",
					...
				}
			}
		});
	
		function onChapterRendererSample1( cuePoint, $chapterBox ){
			$chapterBox.find('.k-title').text( 'custom: ' + cuePoint.text );
			$chapterBox.find('.k-description').text( 'custom: ' + cuePoint.customData.desc );
		}
</script>
